JavaScript学习记录-日期表示

发表于 2019-01-13

ECMAScript中有Date类型用来处理时间和日期。Date类型内置了一系列获取和设置日期时间信息的方法。

Date类型

创建一个日期对象，用new运算符和Date构造方法（构造函数）。

var test = new Date();

在调用没有参数的Date构造方法时候，新建的对象会自动获取当前的时间日期。

ECMAScript提供了两个方法：(1)Date.parse()，(2)Date.UTC()。

Date.parse()方法

接收一个表示日期的字符串参数，然后根据这个字符串返回相应的毫秒数。ECMA-262没有定义Date.parse()应该支持哪种日期格式，因此方法的行为因实现而异，因地区而异。默认通常接收的日期格式：

月/日/年，如6/13/2011；

英文月名 日，年，如 May 25, 2004;

英文星期几 英文月名 日 年 时：分：秒 时区，如 Tue May 25 2004 00:00:00 GMT-070

alert(Date.parse("7/23/2018")); 
//返回 1970/1/1 午夜距离该日期时间的毫秒数。没有参数或者日期错误返回NaN

若要输出指定的日期，则可以将Date.parse()传入Date构造方法中。

var test = new Date("7/23/2018");//直接传入，Date.parse()后台被调用
alert(test);

但不同浏览器所返回的值却不一样，譬如chrome：Mon Jul 23 2018 00:00:00 GMT+0800 (中国标准时间)

UTC() 方法

可根据世界时返回 1970 年 1 月 1 日 到指定日期的毫秒数。Date.UTC(year,month,day,hours,minutes,seconds,ms)，其中year，month，day是必须的。（此处为w3c中所述，但实际在chrome中实验，若只有year一个参数仍能返回数值）当参数传递错误，那么就会出现负值或者NaN等非法信息。其输出指定日期和Date.parse()方法相同。

通用的方法

Date类型也重写了toLocaleString()、toString()和valueOf()方法。

var test = new Date(Date.UTC(2018,7, 24, 9, 0, 0));
alert('toString:' + test.toString());
//输出结果toString:Fri Aug 24 2018 17:00:00 GMT+0800 (中国标准时间)        
alert('toLocaleString:' + test.toLocaleString());
//按本地格式输出，toLocaleString:2018/8/24 下午5:00:00

日期格式化方法

Date类型还有一些专门用于将日期格式化为字符串的方法。

var test= new Date();

alert(test.toDateString());                         //以特定的格式显示星期几、月、日和年
alert(test.toTimeString());                        //以特定的格式显示时、分、秒和时区
alert(test.toLocaleDateString());               //以特定地区格式显示星期几、月、日和年
alert(test.toLocaleTimeString());               //以特定地区格式显示时、分、秒和时区
alert(test.toUTCString());                         //以特定的格式显示完整的UTC日期。
(5)组件方法

组件方法是为单独获取想要的各种时间/日期而提供的方法。需要注意的是这些方法中，有带UTC的，有不带UTC的。

UTC日期指的是在没有时区偏差的情况下的日期值。

alert(test.getTime());                              //获取日期的毫秒数，和valueOf()返回一致

alert(test.setTime(100));                           //以毫秒数设置日期，会改变整个日期

alert(test.getFullYear());                          //获取四位年份

alert(test.setFullYear(2012));                      //设置四位年份，返回的是毫秒数

alert(test.getMonth());                             //获取月份，没指定月份，从0开始算起

alert(test.setMonth(11));                           //设置月份

alert(test.getDate());                              //获取日期

alert(test.setDate(8));                             //设置日期，返回毫秒数

alert(test.getDay());                               //返回星期几，0表示星期日，6表示星期六

alert(test.setDay(2));                              //设置星期几

alert(test.getHours());                            //返回时

alert(test.setHours(12));                          //设置时

alert(test.getMinutes());                          //返回分钟

alert(test.setMinutes(22));                        //设置分钟

alert(test.getSeconds());                          //返回秒数

alert(test.setSeconds(44));                        //设置秒数

alert(test.getMilliseconds());                     //返回毫秒数

alert(test.setMilliseconds());                     //设置毫秒数

alert(test.getTimezoneOffset());                   //返回本地时间和

UTC时间相差的分钟数

以上方法除了最后一个getTimezoneOffset()，其他都具有UTC功能，例如setDate()及getDate()获取星期几，那么就会有setUTCDate()及getUTCDate()，表示世界协调时间。
